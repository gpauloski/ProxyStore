    Redis backend class.

    Parameters:

    • name (str) –

      Name of the store instance.

    • hostname (str) –

      Redis server hostname.

    • port (int) –

      Redis server port.

    • cache_size (int) –

      Size of LRU cache (in # of objects). If 0, the cache is disabled. The cache is local to the Python process.

    • stats (bool) –

      Collect stats on store operations.
